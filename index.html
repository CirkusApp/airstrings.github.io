Air Strings

  1. Install CLI

    # brew install airstrings

  2. Push Localizable.strings to Google Sheets

    # airstrings push -spreadhseetId <GOOGLE_SPREADSHEET_ID> -path ./en/Localizable.strings -sheetName "es/Localizable.strings"

  3. Translate in Google Sheets

    Share, collaborate, automate etc.

  4. Pull Localizable.strings from Google Sheets

    # airstrings pull -spreadhseetId <GOOGLE_SPREADSHEET_ID> -sheetName "es/Localizable.strings" -path ./es/Localizable.strings
